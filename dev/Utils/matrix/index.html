CPPLS.find_invariant_and_variant_columns — Function
find_invariant_and_variant_columns(M::AbstractMatrix)

Scan each column of M and split them into two index vectors: columns whose entries are all identical (invariant_columns) and columns that contain any variation (variant_columns). Useful for removing zero-variance predictors before fitting.

Example

julia> invariant, variant = find_invariant_and_variant_columns([1 2 2; 1 3 4])
([1], [2, 3])
